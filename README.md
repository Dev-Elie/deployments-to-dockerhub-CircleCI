## Set Up

### Clone repo
```bash
git clone https://github.com/Dev-Elie/deployments-to-dockerhub-CircleCI.git contact-form
```

### Install packages & start services

```bash
cd contact-form
npm install
npm start
```
### Running tests

```bash
npx cypress run --headless
```

Happy Deployment!
